Welcome to Brackets SMP!

What's Brackets ?
Brackets is a modded Minecraft SMP (by EnderPrism and Astrali_) inspired by Alinea SMP. To play it, you first need to get the modpack (download link in the mods tab), and you need to join the server by connecting to "brackets-smp.my.pebble.host".
You need to be whitelisted to play on the server.

Minecraft Infos :
Version : 1.19.2
IP : brackets-smp.my.pebble.host
Modloader : Fabric
Whitelist : OFF
