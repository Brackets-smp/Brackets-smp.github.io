Welcome to Brackets SMP!

What's Brackets ?
Brackets is a modded Minecraft SMP (by EnderPrism and Astrali_GD) inspired by Alinea SMP. To play it, you first need to get the modpack (download link below)
and you need to join the server by connecting to brackets-smp.my.pebble.host
You need to be whitelisted to play on the server.

How to join Brackets?
As I said earlier, you need to connect to the Minecraft server using the following IP, brackets-smp.my.pebble.host
But don't forget to have all the mods while joining!
You may have seen that you need to be whitelisted to enter.

How to be whitelisted?
It's simple, you just need to join the Discord server and create a ticket !
A moderator will ask you questions to see if you can play (if you're not toxic, if you know the server's rules, etc) and normally you will be whitelisted !
